# 测试数据集说明

## 重要声明

本目录下的 CSV 文件是**基于公开报道中描述的数据异常模式构造的模拟测试数据**，不是从论文原始 PDF 或 Source Data 中提取的真实数值。

## 为什么没有真实原始数据？

1. Nature 论文的 **Source Data 文件需要订阅权限**才能下载（尝试下载返回 AccessDenied）
2. 被质疑的具体图表数据（Figure 4c、4f、6g）嵌入在论文 PDF 的图片中，需 OCR 或人工抄录
3. Supplementary Tables XLSX（已下载到 `tongji_supplementary.xlsx`）包含的是基因组学数据，不包含被质疑的实验表格

## 数据来源和依据

以下每个案例的数据模式均来自公开报道中的具体描述：

### 案例1：同济大学王平团队 Nature (2024.11)
- 论文：Human HDAC6 senses valine abundancy to regulate DNA damage
- DOI: https://doi.org/10.1038/s41586-024-08248-5
- 处理结果：2026.5.6 同济大学通报认定学术不端，免职
- 来源报道：每日经济新闻、观察者网、知社学术圈、PubPeer

**Figure 4c 异常**（来源：观察者网、PubPeer用户Idas washingtonia）：
- 第3列减第4列恒等于 **0.3**（35个数据点完全满足）
- 第4列加 **2.15%** 等于第8列对应值
- 黄色区域小数点后第二位全是4，绿色区域全是5

**Figure 4f 异常**（来源：每日经济新闻）：
- 70个百分比值全部恰好只有 **1位小数**
- 实际底层数据举例：
  - 27÷9000×100%=**0.3%**
  - 15÷3000×100%=**0.5%**
  - 24÷4000×100%=**0.6%**
  - 27÷3000×100%=**0.9%**
  - 35÷5000×100%=**0.7%**

**Figure 6g 异常**（来源：每日经济新闻）：
- 末位数字"5"出现 **212次**，数字"6"仅 **16次**
- 另一组2400个数据中末位"5"出现频率畸高

**小鼠体重数据**（来源：耿同学视频）：
- 共196只小鼠体重数据
- 仅 **1只**的末位数字为0（正常应约10%）
