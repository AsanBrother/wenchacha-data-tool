
## 1. Product Overview
服务于高等教育学生的论文数据检验工具，提供数据造假检测、问题数据自动修复和仿真数据生成功能。
- 目标用户：高等教育学生、研究人员、论文审稿人
- 核心价值：帮助用户识别论文中可能存在的数据问题，学习正确的统计方法，生成符合学术规范的仿真数据

## 2. Core Features

### 2.1 User Roles
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| Normal User |无需注册| 使用所有核心功能，上传数据，查看报告 |

### 2.2 Feature Module
1. **首页**：功能导航，语言切换，使用说明
2. **数据检测页面**：数据上传，多种统计检验，可视化报告
3. **数据修复页面**：问题数据智能修复，保持统计结论不变
4. **数据生成页面**：自定义参数，多种分布，导出数据

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 首页 | 导航栏 | 中英文切换，功能导航 |
| 首页 | Hero区域 | 产品介绍，核心功能亮点 |
| 首页 | 使用说明 | 各功能模块的简要说明 |
| 数据检测页面 | 数据上传 |支持CSV/Excel/手动输入数据|
| 数据检测页面 | 统计检验 | 末位数字分布检验、卡方检验、Benford定律、重复数据检测、等差数列检测 |
| 数据检测页面 | 可视化报告 | 图表展示检验结果，问题标记，详细报告下载 |
| 数据修复页面 | 问题标记 | 显示检测出的问题数据 |
| 数据修复页面 | 智能修复 | 自动替换不合理数据，保持原统计特征（均值、方差等）|
| 数据修复页面 | 前后对比 | 修复前后数据对比，统计量对比 |
| 数据生成页面 | 参数配置 | 选择分布类型、均值、方差、样本量等 |
| 数据生成页面 | 预览与导出 | 数据预览，导出CSV/Excel |

## 3. Core Process
用户进入首页选择功能模块 -&gt; 上传或配置数据 -&gt; 执行检测/修复/生成 -&gt; 查看结果并导出

```mermaid
flowchart TD
    A[首页] --&gt; B{选择功能}
    B --&gt;|数据检测| C[上传数据]
    B --&gt;|数据修复| D[上传问题数据]
    B --&gt;|数据生成| E[配置参数]
    C --&gt; F[执行统计检验]
    D --&gt; G[执行智能修复]
    E --&gt; H[生成仿真数据]
    F --&gt; I[查看报告]
    G --&gt; J[对比结果]
    H --&gt; K[预览数据]
    I --&gt; L[导出报告]
    J --&gt; M[导出修复数据]
    K --&gt; N[导出数据]
```

## 4. User Interface Design
### 4.1 Design Style
- 主色调：深蓝色（#165DFF）配合白色背景，体现学术严谨性
- 辅助色：浅蓝色渐变用于强调区域，红色用于警告标记
- 按钮风格：圆角矩形，轻微阴影，悬停时有上浮效果
- 字体：Inter 作为主要字体，中文字体使用思源黑体
- 布局风格：卡片式布局，充足留白，清晰的视觉层次
- 图标：使用 Lucide React 图标库，风格统一简洁

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 首页 | Hero区域 | 大号标题，简洁说明文字，功能卡片网格，图标配合文字 |
| 数据检测页面 | 数据上传 |拖拽上传区域，文件类型提示，预览表格|
| 数据检测页面 | 检验配置 |复选框选择检验方法，参数设置卡片|
| 数据检测页面 | 结果展示 |图表组件，问题数据高亮，详细报告面板|
| 数据修复页面 | 修复界面 |左右分栏对比，滑动条调整修复强度，确认按钮|
| 数据生成页面 | 参数配置 |表单式布局，实时预览区域，分布类型选择器|

### 4.3 Responsiveness
- Desktop-first 设计，自适应各种屏幕尺寸
- 移动端优化导航和交互元素，确保在手机上也能正常使用
- 触摸友好的按钮和表单元素，尺寸适合手指操作

### 4.4 3D Scene Guidance
本项目不涉及3D场景。
