## ⚡ 快速部署步骤

### 步骤1：配置阿里云CLI（只需配置一次）

```bash

chmod +x setup-alicloud.sh

# 运行配置脚本
./setup-alicloud.sh
```

如果看到 `✓ 连接成功！`，说明配置正确！

### 步骤2：部署网站


## ❓ 常见问题

### Q：如果提示权限不足？
**A**：请在阿里云RAM控制台给这个子账号添加权限。

### Q：部署后访问是403？
**A**：请检查存储桶权限是否为「公共读」。

### Q：部署后访问是404？
**A**：请检查静态网站托管是否配置正确（首页设置为index.html）。
